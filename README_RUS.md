# s21_matrix

Собственная библиотека для работы с матрицами на языке C.

Проект реализует основные операции линейной алгебры, включая арифметику матриц, вычисление определителя, транспонирование, нахождение матрицы алгебраических дополнений и обратной матрицы.

## Обзор

Матрицы являются одной из фундаментальных структур данных в вычислительной математике и используются в компьютерной графике, моделировании, обработке данных, машинном обучении и численных методах.

В рамках проекта реализован динамический тип матрицы и набор базовых математических операций без использования сторонних библиотек.

## Представление матрицы

Матрица хранится в виде динамически выделяемого двумерного массива:

```c
typedef struct matrix_struct {
    double **matrix;
    int rows;
    int columns;
} matrix_t;
```

Размер матрицы определяется во время выполнения программы.

## Возможности

### Управление матрицами

Создание и освобождение памяти:

```c
s21_create_matrix()
s21_remove_matrix()
```

### Сравнение матриц

Сравнение матриц с учётом точности вычислений:

```c
s21_eq_matrix()
```

Точность сравнения — до 1e-7.

### Арифметические операции

Сложение и вычитание:

```c
s21_sum_matrix()
s21_sub_matrix()
```

Умножение на число:

```c
s21_mult_number()
```

Умножение матриц:

```c
s21_mult_matrix()
```

### Преобразования

Транспонирование матрицы:

```c
s21_transpose()
```

### Операции линейной алгебры

Вычисление определителя:

```c
s21_determinant()
```

Нахождение матрицы алгебраических дополнений:

```c
s21_calc_complements()
```

Вычисление обратной матрицы:

```c
s21_inverse_matrix()
```

Обратная матрица вычисляется по формуле:

```text
A⁻¹ = (1 / det(A)) × Cof(A)ᵀ
```

## Используемые алгоритмы

Для реализации операций линейной алгебры используются классические алгоритмы вычислительной математики.

### Вычисление определителя

Определитель вычисляется рекурсивным методом разложения Лапласа по первой строке матрицы.

Особенности реализации:

- прямые формулы для матриц 1×1 и 2×2;
- построение миноров для матриц большего размера;
- рекурсивный расчёт определителей миноров;
- учёт знака алгебраического дополнения;
- динамическое создание временных матриц.

Формула:

det(A) = Σ (-1)^j · a₀ⱼ · M₀ⱼ

где M₀ⱼ — определитель соответствующего минора.

Такой подход делает реализацию наглядной и математически близкой к классическому определению определителя.

## Обработка ошибок

Все операции выполняют проверку:

* корректности матрицы;
* совместимости размеров;
* существования определителя;
* возможности нахождения обратной матрицы.

Для информирования о результате используются стандартизированные коды возврата.
